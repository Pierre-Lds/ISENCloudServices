# ISENCloudServices

# Proj'ISEN

ISENCloudServices is an ordonnancer to manage and book ressources.

Node v18.12.1 / Pug / MongoDB

## Requirements
- Node v12 min

## Setup
- Install NodeJS
- Go to the root and run :
```bash
npm install
```

## Run the application

- Go to the root and run :
```bash
node index.js
```
